## üìÇ Project Showcase

### 1. [Single-Page Semantic HTML CV](./Single-Page%20CV/)
- **Tech:** Pure HTML5
- **Focus:** Document structure, SEO Meta Tags, Open Graph integration, and accessibility.
- **Goal:** Build a professional CV using strictly semantic markup without any styling.

### 2. [Basic Multi-Page HTML Website](./Basic%20HTML%20Website/)
- **Tech:** Pure HTML5
- **Focus:** Multi-page navigation, global `<nav>`, and structural hierarchy.
- **Goal:** Create an interlinked website (Home, Projects, Articles, Contact) practicing internal linking.

### 3. [Responsive Changelog Component](./Changelog%20Component/)
- **Tech:** HTML5, CSS3 (Flexbox/Grid)
- **Focus:** Timeline layouts, responsive design, and modern UI aesthetics.
- **Goal:** Build a sleek release notes component that adapts from mobile to desktop.

### 4. [Responsive Testimonial Cards](./Testimonial%20Cards/)
- **Tech:** HTML5, CSS3 (Grid)
- **Focus:** Card-based UI, layout variations, and hovering micro-interactions.
- **Goal:** Design premium testimonial sections with varied alignment styles.

### 5. [Responsive Image Grid Layout](./Image%20Grid%20Layout/)
- **Tech:** HTML5, CSS3 (Grid)
- **Focus:** Asymmetric grid placement (`grid-column`/`grid-row` spans) and `object-fit`.
- **Goal:** Create a professional-grade photography gallery with a "Featured" image style.

### 6. [Accessible Form UI](./Accessible%20Form%20UI/)
- **Tech:** HTML5, CSS3
- **Focus:** **WCAG 2.1 AA Compliance**, ARIA roles, focus management, and high contrast.
- **Goal:** Develop a production-ready account registration form optimized for screen readers and keyboard users.

### 7. [Simple Tabs Component](./Tabs/)
- **Tech:** HTML5, CSS3, Vanilla JavaScript
- **Focus:** DOM manipulation, event handling, and accessible interactive states.
- **Goal:** Build a functional tabs interface using pure JS with full keyboard navigation support.

### 8. [Cookie Consent Banner](./Cookie%20Consent/)
- **Tech:** HTML5, CSS3, Vanilla JavaScript
- **Focus:** `localStorage` persistence, conditional rendering, and fixed positioning.
- **Goal:** Create a privacy banner that remembers user consent across browser sessions.

### 9. [Age Calculator](./Age%20Calculator/)
- **Tech:** HTML5, CSS3, JavaScript (ES6), **Luxon**, **Flatpickr**, **Vite**
- **Focus:** npm package integration, complex date math, and third-party library initialization.
- **Goal:** Build a professional tool that calculates exact age (Years, Months, Days) using external logic and custom pickers.

### 10. [GitHub Random Repository Finder](./GitHub%20Random%20Repository/)
- **Tech:** HTML5, CSS3, Vanilla JavaScript, **GitHub API**
- **Focus:** API integration (Fetch/Async-Await), state management (Loading/Error/Empty), and random selection.
- **Goal:** Build an intermediate-level app to discover trending repositories via asynchronous request handling.

### 11. [Instagram Story Clone](./Instagram-Story-Clone/)
- **Tech:** HTML5, CSS3, Vanilla JavaScript, **Local Storage API**
- **Focus:** 24hr expiration logic, Base64 image storage, fullscreen modal UI, and progress timers.
- **Goal:** Build a complex, client-side social feature with persistent data and interactive viewing.

## üöÄ How to Run These Projects

### For Pure HTML/CSS Projects:
Simply navigate to the project folder and open `index.html` in any modern web browser.

### For JavaScript Projects:
1. Navigate to the project folder (e.g., `Tabs`, `Cookie Consent`, `GitHub Random Repository`, or `Instagram-Story-Clone`).
2. Open `index.html` in your browser.

### For the Age Calculator (npm required):
1. Navigate to the `Age Calculator` folder.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open the provided local URL (usually `http://localhost:5173`).
