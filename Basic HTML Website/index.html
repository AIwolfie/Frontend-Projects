Welcome to My Website

Hello! This is a beginner-level project designed to demonstrate the power of Semantic HTML. This website consists of multiple pages linked together, organized with clean markup and a logical structure.

Feel free to explore my projects, read my articles, or get in touch through the contact page.

About This Project

The goal of this multi-page website is to practice document structure without relying on CSS for layout. By using tags like <header>, <main>, and <footer>, we ensure that the content is accessible and well-organized.
